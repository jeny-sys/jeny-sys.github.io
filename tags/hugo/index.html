关于这个博客

这个博客使用 Hugo 静态站点生成器构建，采用极简主义设计风格。主要特点包括：

  • 双栏布局：左侧导航，右侧内容
  • 极简风格：大量留白，专注于内容本身
  • 响应式设计：适配各种设备屏幕

技术栈

  • Hugo：静态站点生成器
  • HTML5/CSS3：前端技术
  • JavaScript：交互功能
  • GitHub Pages：托管服务
